# Code Of Conduct

Dave OS should be a useful, generous technical space.

## Expected Behavior

- Be direct, kind, and specific.
- Assume good intent, but accept correction quickly.
- Critique code and ideas, not people.
- Make room for contributors with different experience levels.
- Prefer clear tradeoffs over status games.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Sexualized, discriminatory, or demeaning language.
- Publishing private information without permission.
- Repeatedly derailing technical discussions after maintainers redirect.

## Reporting

Report problems privately to the maintainer listed in [SECURITY.md](./SECURITY.md). Maintainers may remove comments, close issues, block users, or take other reasonable action to keep the project healthy.
