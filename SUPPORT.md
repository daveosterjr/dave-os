# Support

Use GitHub issues for:

- starter generation bugs;
- dependency install failures;
- docs gaps;
- integration requests that fit the default stack.

Use discussions, once enabled, for:

- architecture questions;
- template ideas;
- examples of generated apps;
- comparisons with other starters.

Please include the command you ran, Node/npm versions, and the generated app name when reporting generator issues.
