# Development

## Daily Loop

```bash
npm run dev
```

This starts local services and the Next app with labeled output.

## Local Services

- Postgres: `localhost:5432`
- Redis: `localhost:6379`
- Mailpit SMTP: `localhost:1025`
- Mailpit UI: `http://localhost:8025`

## Useful Commands

```bash
npm run db:start
npm run db:push
npm run db:studio
npm run trigger:dev
npm run stripe:listen
npm run verify
```
