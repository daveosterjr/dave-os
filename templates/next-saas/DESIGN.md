# __APP_TITLE__ Design

## Product Feel

Start restrained and useful. This starter is built for dense product work, not a generic landing page.

## UI Rules

- Tailwind is the styling layer.
- Local shadcn-style primitives live in `components/ui`.
- Use lucide icons for icon buttons.
- Cards are for repeated items, modals, and framed tools.
- Do not nest cards inside cards.
- Keep settings, billing, and admin surfaces quiet, scan-friendly, and efficient.
- Keep auth pages custom to this product. Do not use default Clerk screens.

## Tokens

The starting palette is neutral and semantic. Change it per product, but keep tokens named by purpose.
